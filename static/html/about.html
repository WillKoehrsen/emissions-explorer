About

The Emissions Explorer is a tool designed to illustrate the power of simple visiualizations to make data come to life. This application was built for HackCWRU V. Currently, the Emissions Explorer consists of three parts: a world map showing historical data for all available countries from 1850-2014; a chart showing the history of the top 15 emitters; and a socio-economic and emissions comparison of the top 15 countries. The application uses d3, a visualization library built on Javascript, and Python was used to format the data. A list of references and data sources is provided on the left. The website is hosted on an Amazon S3 storage bucket with the domain name purchased from Hover. I welcome questions, feedback, and discussion and can be reached onTwitter.

Data

Data for this project came from a number of different sources: CAIT Climate Data, World Bank, and GapMinder (helpfully curated by Tom Carden) with links on the left. All the Jupyter Notebooks for formatting the data are available on the project GitHub with this Notebook a helpful look at the process of data cleaning. I believe in the benefits of free and open data and also in making that data understandable. Having the data available is a good start, but if it remains in a spreadsheet, it will not inspire anyone to action. My goals in this project were to explore one method for presenting data, learn some new skills, visualize one crucial public dataset, and hopefully inspire others to start deriving knowledge from data.
